Первый и фундаментальный шаг — узлы должны находить друг друга автоматически, без ввода IP вручную.

## Выбранный подход: mDNS

**mDNS (Multicast DNS)** — каждый узел объявляет себя в локальной сети под именем вида `void-connect.local`. Остальные узлы слышат это объявление и автоматически добавляют его в список участников.

Rust-библиотека: `mdns-sd`

## Как это работает

1. При запуске программы узел регистрирует себя через mDNS
2. Одновременно начинает слушать объявления других узлов
3. Формируется живой список участников сети (peer list)
4. Если узел отключается — он пропадает из списка

## Дополнительно: UDP broadcast

Как резервный вариант можно добавить UDP broadcast — узел отправляет пакет на широковещательный адрес (`255.255.255.255`) со своим IP и портом. Менее элегантно, но работает там, где mDNS заблокирован.

## Структура записи об узле

```json
{
  "id": "публичный_ключ_узла",
  "name": "Vasya",
  "ip": "192.168.1.5",
  "port": 7700,
  "services": ["chat", "storage", "web"],
  "last_seen": 1234567890
}
```
