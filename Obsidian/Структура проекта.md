Проект написан на Rust и организован как **Cargo workspace** — несколько связанных крейтов в одном репозитории.

## Дерево файлов

```
void-connect/
├── Cargo.toml                  # Workspace манифест
├── README.md
│
├── crates/
│   │
│   ├── core/                   # Общие типы и утилиты
│   │   ├── src/
│   │   │   ├── lib.rs
│   │   │   ├── identity.rs     # Ключи, ID пользователя, подписи
│   │   │   ├── profile.rs      # Структура профиля пользователя
│   │   │   ├── message.rs      # Общие типы сообщений
│   │   │   └── error.rs        # Общие типы ошибок
│   │   └── Cargo.toml
│   │
│   ├── discovery/              # [[Обнаружение узлов]]

│   │   ├── nat.rs          # UPnP, hole punching, STUN

│   │   │   ├── udp_broadcast.rs# Резервный метод обнаружения
│   │   │   └── peer_list.rs    # Живой список известных узлов
│   │   └── Cargo.toml

│   ├── network/                # Сетевой слой
│   │   ├── src/

│   │   │   ├── site_registry.rs# Каталог сайтов в сети

    │   ├── main.rs             # Точка входа GUI (eframe)

    │       └── status_bar.rs   # Статус подключения, режим (LAN/Global)
    └── Cargo.toml
```

## Зависимости между крейтами

```
app
 ├── discovery
 ├── chat  ──────── network
 ├── storage ────── network
 ├── web ─────────── storage
 ├── reputation
 └── все крейты используют: core, crypto, db
```

## Ключевые внешние зависимости

|Крейт|Библиотека|Назначение|
|---|---|---|
|discovery|`mdns-sd`|mDNS обнаружение|
|network|`tokio`, `tokio-tungstenite`|Async runtime, WebSocket|
|web|`axum`|HTTP-сервер|
|crypto|`libsodium` / `sodiumoxide`|Шифрование и подписи|
|db|`sqlx`, `sqlite`|База данных|
|storage|`sha2`|SHA-256 хэши|
|ui|`egui`, `eframe`|Графический интерфейс|

## Связанные компоненты

- [[Обнаружение узлов]] → `crates/discovery`
- [[Общий чат]] / [[Личный чат]] → `crates/chat`
- [[Общее хранилище данных]] → `crates/storage`
- [[Поднятие простых сайтов]] → `crates/web`
- [[Внутренний DNS]] → `crates/web/src/dns.rs`
- [[Система репутации]] → `crates/reputation`
- [[Безопасность сети]] → `crates/crypto`, `crates/network`
- [[Глобальная сеть]] → `crates/discovery/src/bootstrap.rs`, `crates/discovery/src/nat.rs`
- Интерфейс → `ui/` (egui)
