# Void Connect

Децентрализованная P2P сеть — мини-интернет внутри локальной сети.

## Быстрый старт

```bash
# Клонируй репозиторий и войди в папку
cd void-connect

# Запусти (покажет логи обнаружения узлов)
RUST_LOG=debug cargo run

# На втором компьютере в той же сети — то же самое
# Узлы найдут друг друга через mDNS автоматически
```

## Структура проекта

```
crates/
  core/        # Общие типы: NodeId, PeerInfo, NetworkMessage
  crypto/      # Ключи, подписи, E2E шифрование (libsodium)
  db/          # SQLite через sqlx
  discovery/   # ✅ mDNS + UDP broadcast обнаружение узлов
  network/     # TCP/WebSocket соединения
  chat/        # Общий и личный чат
  storage/     # P2P обмен файлами по схеме чанков
  web/         # HTTP-сервер, хостинг сайтов, DNS .void
  reputation/  # Система репутации
app/           # Точка входа
```
